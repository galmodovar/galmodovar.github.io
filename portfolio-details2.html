TrackIt

Project information

  • Tech Used: React | MaterialUI | Django | Chartjs
  • Sample Login: U: John123 PW: me

This is an application that allows users to track where they are in the job hunting process.

A user can register an account, create an entry for a job application with fields for items like company, job title, industry, status as well as the ability to add notes about the job and what tech stack the job is using.

For this app I wanted to try different things like using Material UI and ChartJS. About halfway into the app I also decided on trying to implement a drag and drop feature for the application entries. While I got the feature to work, it does not currently save state so on refresh the entries go back to the first column which is the initial state. Because this project was part of the capstone requirement for Nashville Software School and not a passion project I likely will not go back to build the feature to save state.

I currently use this app more to test out new or interesting features (like the drag and drop and ChartJS)

If I were to start the app over I would have done a better job planning and fully committing to the idea as planned. The models were built in Django and currently have several fields that may not actually be needed. If I wanted to use a drag and drop then there really wasn't a need for fields like stage in status in the app form.
