College Football Weekly Pick 'Em

Project information

  • Tech Used: React | CSS | ESPN's API
  • Sample Login: U: JOHN@DOE.COM PW: me

This application is similar to a traditional fantasy football game, only instead of users picking players and getting a score for the week, users will pick college football teams for the week. The score will be calculated by adding up the final scores for each team on the roster.

Users can register an account,as well as add and delete teams for the week. The scores update in real time and can check how they're doing against other users by checking the leaderboard.

I was able to learn a lot by creating an ERD to help track data relationships and utiliizing ESPN's external API to bring in the college football information. I got to practice a lot with different array methods to get the actual data that was needed. The app saves the team to a local json database file by using the fetch api with a simple POST method. The method saves the userId, the current week and the teamID (which is taken directly from ESPN's team ids). I really enjoyed learning how to use an external API and realized how important it is to test and track data flow. I do not believe this project would scale well without building out some functionality to create private groups or challenges so users can play with their friends.
